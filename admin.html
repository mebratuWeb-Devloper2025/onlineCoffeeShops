Role-Based Permissions

Detailed overview of access levels for each admin role

System Administrator

Overall Control
Create/edit/delete all admin accounts
Access ALL system features and settings
Backup and restore system data
View all logs and activity history
Emergency system overrides
Manage payment gateway settings
Can impersonate any user role

Financial Controller

Price & Reports
Update product prices (coffee items)
Set special offers/discounts
View daily/weekly/monthly sales reports
Generate profit/loss statements
Process refunds
Track revenue and expenses
Export financial data (Excel/PDF)

Restrictions

Cannot create/delete admin accounts
Cannot modify system settings
Cannot manage menu items (only pricing)

Product Manager

Menu Management
Add new coffee/products to menu
Edit existing menu items
Delete/archive menu items
Update product descriptions
Upload product images
Manage product categories
Set product availability status

Restrictions

Cannot change product prices
Cannot view financial reports
Cannot access transaction details
Cannot modify payment settings
Cannot access admin management
